## AI is Your Best Friend

Use AI tools — Claude, GitHub Copilot, ChatGPT, whatever works for you. AI is a core part of modern software development and is explicitly encouraged in this assessment. Use it to plan, scaffold, debug, write tests, and document your work.

Just make sure you understand what it produces, remember how you use it will be marked.
